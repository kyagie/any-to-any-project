# A2App (any-to-any) 

Converts numbers between any bases
https://any-to-any.web.app/#/
Live App here. [Any To Any App Live Demo](https://any-to-any.web.app/#/)

## Install the dependencies
```bash
npm install
```

## Build
```bash
quasar build -m pwa
```
